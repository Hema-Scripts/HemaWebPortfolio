Raspberry Pi Mini Server

❮ ❯

Description

I built a low-cost Raspberry Pi-based mini server to provide real-time monitoring of system performance, including detail about resource usage and health. It is a flexible server that can be converted into a VPN, ad blocker, private cloud, or private NAS. This makes it a very good solution for home networking to small business needs.

Working

Monitors key metrics such as CPU load, memory usage, and disk space in real-time. The server features a built-in Uninterruptible Power Supply (UPS) to ensure continuous operation, complemented by an OLED display that provides immediate feedback on system performance.

Components

  • Raspberry Pi 4 Model B
  • MicroSD Card (32GB or larger)
  • Power Supply (5V, 3A)
  • USB HDD or SSD for storage
  • OLED Display (optional)
  • Cooling Fan (optional)
  • Network cable (Ethernet)

Usage

The Raspberry Pi mini server can serve multiple functions:

  • VPN Server: Securely connect to your home network from anywhere in the world, ensuring privacy while browsing.
  • Ad Blocker: Utilize software like Pi-hole to block unwanted ads and trackers across all devices on your network.
  • Private Cloud Storage: Set up your own cloud solution using Nextcloud, allowing you to store and access files securely from any location.
  • Private NAS (Network Attached Storage): Store and manage your media files, documents, and backups with ease, providing a centralized location for your data.

Benefits

The Raspberry Pi Mini Server is a cost-effective and energy- efficient solution that can function as a VPN server, ad blocker, private cloud storage, and NAS. Its versatility allows for easy customization, making it ideal for both personal use and educational purposes. With extensive community support, users can access resources and tutorials to enhance their learning experience.

WiFi Deauther

❮ ❯

Description

The WiFi Deauther is a powerful tool designed for network security testing and ethical hacking. Built using the NodeMCU ESP8266, it enables users to analyze and identify vulnerabilities within WiFi networks by deauthenticating devices, which helps assess the security of wireless connections. This project serves as an educational platform for understanding network protocols and enhancing cybersecurity awareness, making it an essential resource for security enthusiasts and professionals.

Working

The WiFi Deauther sends deauthentication packets to a targeted WiFi network, disconnecting devices attempting to connect. This allows users to simulate attacks and assess network security vulnerabilities, helping to improve defenses against potential threats. It enables analysis of device responses and testing of various attack vectors on WiFi protocols.

Components

  • NodeMCU ESP8266
  • Power Supply
  • Breadboard and Jumper Wires
  • Computer
  • WiFi Adapter

Usage

The WiFi Deauther is used for penetration testing and security assessments of wireless networks. It helps identify vulnerabilities by disconnecting devices, allowing network administrators to evaluate their defenses. Additionally, it's useful for educational purposes, demonstrating the importance of WiFi security and the potential risks of weak protocols.

Benefits

  • Vulnerability Assessment: Identifies weaknesses in WiFi networks, helping to enhance security measures.
  • Network Testing: Assists network administrators in testing the effectiveness of their security protocols.
  • Educational Tool: Serves as a practical demonstration for understanding wireless security risks and the importance of robust protections.
